# Social Media Ad Campaign Analysis

## Overview
Built an end-to-end data pipeline using Databricks and Power BI to analyze 100K+ ad campaign records. Implemented the Medallion Architecture (Bronze → Silver → Gold) to transform raw data into actionable business insights.

---

## Architecture
- Bronze: Raw data ingestion (unchanged dataset)
- Silver: Cleaned and transformed data
- Gold: Aggregated SQL views for reporting
- Visualization: Power BI dashboards connected to Databricks

---

## Key Work
- Performed data validation and exploratory checks on raw dataset
- Cleaned and transformed data using SQL in Databricks
- Decoded categorical variables using CASE statements
- Created derived KPIs such as CTR and CVR using safe calculations
- Designed optimized SQL views for reporting and analysis
- Connected Power BI to Databricks using native connector
- Built an interactive multi-page dashboard for campaign insights

---

## Dashboard Highlights
- Daily performance tracking with KPI cards and trend analysis
- Audience segmentation based on age groups and gender
- Campaign performance by device type and category
- Summary view with structured reporting and key metrics
